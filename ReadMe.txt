This program saves name and address information in the SaveInfo.txt file, and it retrieves a password to
decrypt from the password.txt file. It also uses cpsclogo.png as the shopping window header, savelogo.png as the save button image and clearlogo.png as the clear button image. All of these files should be kept in the same directory as the other program files.
